# 배부룩 (Babule) Frontend

맛집 추천 서비스 프론트엔드

## 기술 스택

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS

## 페이지 구조

| 경로 | 페이지 |
|------|--------|
| `/` | 홈 |
| `/map` | 맛집 지도 |
| `/roulette` | 메뉴 추천 |
| `/list` | 맛집 목록 |
| `/my` | 마이페이지 |

## 시작하기

```bash
npm install
npm run dev
```

`http://localhost:3000`에서 확인할 수 있습니다.
